Installation

What is CSS testing?

Tests that make sure your visual layout is as sturdy as your javascript. After your page looks the way you want it to, create a few test cases of key styles. During your automated build, run those tests to ensure everything looks as it should.

Find out more at csste.st»

What is Hardy?

A command-line tool that simplifies CSS testing by using Cucumber and makes it simple to include in your automated build process. There's also a chrome extension to simplify making test cases. It runs on Node.JS and uses Selenium.
